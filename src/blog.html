  • Question-1: What are the different ways to select an element in the DOM?

    There are few ways of selecting an element in the DOM. The are document.getElementById(), document.getElementsByClassName(), document.querySelector(), document.querySelectorAll()

  • Question-2: What is the difference between innerHTML, innerText, and textContent?

    innerHTML retrieves HTML content including tags, innerText returns visible text only, and textContent gives all text content, including hidden elements.

  • Question-3: What is event delegation in the DOM?

    Event delegation is when an event listener is added to a parent element, which listens for events on child elements, improving efficiency.

  • Question-4: What is event bubbling in the DOM?

    Event bubbling is when an event starts at the target element and bubbles up to its parent elements in the DOM hierarchy.

  • Question-5: How do you create, add, and remove elements using JavaScript?

    To create element we can use document.createElement(), to add element we can use appendChild(), and to remove element we can use removeChild()
